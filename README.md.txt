# Exploratory Data Analysis on Titanic Dataset  

This project was completed as part of my **CodeAlpha Data Analytics Internship**.  
The main objective was to perform **Exploratory Data Analysis (EDA)** on the Titanic dataset  
to identify trends, patterns, and key factors that influenced passenger survival.

---

## ЁЯОп Objective  
The purpose of this project is to analyze the Titanic dataset using Python  
to understand how variables such as gender, age, and class affected survival rates.

---

## Tools and Libraries Used  
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- SciPy  

---

##  Project Steps  

1. **Data Loading & Inspection** тАФ Imported and explored the Titanic dataset.  
2. **Data Cleaning** тАФ Handled missing values, removed duplicates, and verified data types.  
3. **Univariate Analysis** тАФ Studied distributions of features like Age, Fare, and Pclass.  
4. **Bivariate Analysis** тАФ Compared Survival with features like Sex, Age, and Pclass.  
5. **Correlation & Visualization** тАФ Created heatmaps and plots to identify relationships.  
6. **Hypothesis Testing** тАФ Performed Chi-square and T-tests for statistical validation.  

---

## ЁЯТб Key Insights  

1. Female passengers had higher survival rates than males.  
2. 1st class passengers were more likely to survive compared to 2nd and 3rd class.  
3. Younger passengers (below ~30 years) had better survival chances.  
4. Families of size 2тАУ4 had higher chances of survival.  
5 Passengers who paid higher fares had better survival rates.  
6. Statistical tests confirmed that class and age significantly influenced survival outcomes.

import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
import seaborn as sns
sns.set(style='whitegrid')
plt.rcParams['figure.dpi'] = 120

# 1. Load
df = pd.read_csv('data/titanic.csv')

# 2. Quick checks

plt.close()

# 5. Bivariate - survival vs class
plt.figure(figsize=(7,5))
sns.countplot(x='Pclass', hue='Survived', data=df)

plt.close()

# 6. Correlation
